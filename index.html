Scientific Image Analysis Platform

Welcome to Aqua Insight

Aqua Insight is a browser-based scientific image analysis platform designed for practical laboratory workflows, aquaculture research, histology, particle detection, morphometric analysis, and other image-based scientific applications.

Open Particle Counter

What Aqua Insight Can Do

Scientific Image Analysis

Analyze scientific images directly in the browser without requiring software installation, backend processing, or external servers.

Particle Counter Analysis

Detect, outline, label, measure, and export particle information using thresholding, connected component labeling, contour detection, and morphological filtering.

Interactive Preview Workflow

Visualize each image processing stage including original image, selected channel, threshold result, and final contour overlay.

Batch Processing

Process multiple images simultaneously using shared parameters and export combined or individual analysis results.
